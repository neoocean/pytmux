스크롤백 · 복사

지난 출력을 되짚어 보고, 필요한 부분만 복사하고, 안전하게 붙여넣습니다. 패널마다 독립적입니다.

스크롤백 보기

copy-mode 로 따로 들어가지 않아도, 패널 위에서 휠을 올리면 바로 지난 출력을 봅니다(alt-scroll on, 기본값). 더 정식으로는 prefix [ 로 스크롤 (복사) 모드에 들어갑니다. 복사 모드 키 스타일은 set mode-keys vi(기본) 또는 emacs 로 고릅니다.

  • 스크롤백 안에서 텍스트를 검색하고 일치 지점으로 점프
  • capture-pane(-S 로 전체 스크롤백) 로 버퍼에 담기
  • clear-history 로 스크롤백 비우기
  • pipe-pane 으로 패널 출력을 외부 명령에 흘려보내기

마우스로 복사

그냥 왼쪽 버튼으로 드래그하면 pytmux 가 드래그를 시작한 패널 안으로만 선택을 가둬, 손을 떼는 순간 OS 클립보드로 자동 복사합니다(mouse-drag-copy, 기본 on). Shift 도 Ctrl+C 도 필요 없고, pytmux 가 그린 패널 외곽선(│)이나 옆 패널 내용은 딸려오지 않습니다. Claude 처럼 마우스를 쓰는 화면 위에서도 드래그가 복사로 갑니다(클릭은 종전대로 버튼/포커스). 자세한 동작과 예외는 6. 마우스로 제어에서 다룹니다.

앱에 마우스 드래그를 넘기고 싶으면 set mouse-drag-copy off — 그때 선택은 Shift+드래그나 위 스크롤백(copy-mode)에서 드래그로 합니다.

붙여넣기

붙여넣기는 bracketed paste 로 한 번에 전달됩니다 — 여러 줄을 붙여도 줄마다 바로 실행되지 않습니다. 붙여넣기 버퍼를 직접 다룰 수도 있습니다.

  • paste-clipboard — OS 클립보드 붙여넣기
  • paste-buffer (prefix ]) — pytmux 붙여넣기 버퍼에서
  • choose-buffer (prefix =, 별칭 list-buffers) — 버퍼 목록에서 골라 붙여넣기
박스드로잉 제거: 분할된 화면을 복사해 붙여넣으면 패널 테두리(│─┼)가 섞여 들어올 수 있습니다. set strip-box-drawing on(기본 ON)이면 붙여넣을 때 이런 박스드로잉 문자를 자동으로 걸러냅니다.

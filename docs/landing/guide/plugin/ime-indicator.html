플러그인 · 입력

ime-indicator

지금 입력 소스가 한글인지 영문인지 화면 우상단 배지로 보여줍니다. 터미널 앱은 IME 상태를 보여주지 않아 한/영 오타가 잦은 것을 줄여 줍니다.

여는 법

명령설명
ime-indicator우상단 한/영 배지 표시 토글

메모

macOS 에선 입력소스 감시 헬퍼(oskbd)를 자식 프로세스로 띄워 상태를 받습니다. 배지는 표시 전용이라 입력 스트림에는 관여하지 않습니다.

끄기 · 제거

:plugins 관리 팝업에서 끄거나(가역), pytmuxlib/plugins/ime-indicator/ 디렉토리를 지우면(delete-to-disable) 명령·자동완성·메뉴 어디에도 나타나지 않고 조용히 사라집니다. 코어는 영향받지 않습니다. ← 플러그인 개요

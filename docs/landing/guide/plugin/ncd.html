ncd

1980년대 Norton Change Directory 를 재현한 디렉토리 전용 트리입니다. 루트부터 현재 디렉토리까지 펼쳐 열리고, 타이핑으로 점프해 Enter 한 번으로 패널을 cd 시킵니다.

여는 법

명령설명
ncd디렉토리 트리 팝업 (별칭 nc)

키동작
↑↓ · PgUp/PgDn · Home/End이동
→ / ←펼치기(지연 로드) / 접기·부모로

메모

Windows 에선 드라이브들이 트리 최상위에 묶여 드라이브 전환도 됩니다. cd 명령의 방언(cmd cd /d vs POSIX)은 셸을 소유한 서버 기준이라 원격 페더레이션에서도 안전합니다. mdir 플러그인의 F10 트리도 이 화면을 재사용합니다.

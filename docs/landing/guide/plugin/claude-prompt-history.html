플러그인 · Claude

claude-prompt-history

Claude 패널에 보낸 프롬프트들을 기억해, 명령 프롬프트 작성 중 직전 프롬프트 미리보기를 띄우고, 팝업에서 시간순으로 훑어볼 수 있게 합니다.

여는 법 · 명령

명령설명
prompt-history프롬프트 히스토리 팝업 (별칭 prompts·ph)
prompt-history-lines미리보기 패널 최대 행수 1~3 (별칭 ph-lines)

메모

: 프롬프트에서 무언가 작성 중이면 위쪽에 직전 프롬프트가 살짝 보입니다(행수는 ph-lines). 히스토리 보존 분량은 서버 옵션으로 영속됩니다.

끄기 · 제거

:plugins 관리 팝업에서 끄거나(가역), pytmuxlib/plugins/claude-prompt-history/ 디렉토리를 지우면(delete-to-disable) 명령·자동완성·메뉴 어디에도 나타나지 않고 조용히 사라집니다. 코어는 영향받지 않습니다. ← 플러그인 개요

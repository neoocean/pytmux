clock

tmux 의 clock-mode 처럼 현재 패널을 큰 블록 숫자 시계로 덮습니다. 셸은 그대로 살아 있고, 표시만 덮는 오버레이입니다.

여는 법

명령설명
clock-mode토글 (prefix t, 우클릭 메뉴에도 있음)
open-clock / close-clock명시적으로 켜기 / 끄기

패널 클릭이나 Shift+Esc 로도 닫습니다.

끄기 · 제거

:plugins 관리 팝업에서 끄거나(가역), pytmuxlib/plugins/clock/ 디렉토리를 지우면(delete-to-disable) 명령·자동완성·메뉴 어디에도 나타나지 않고 조용히 사라집니다. 코어는 영향받지 않습니다. ← 플러그인 개요

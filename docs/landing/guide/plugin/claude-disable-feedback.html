플러그인 · Claude

claude-disable-feedback

Claude Code 가 띄우는 시작 팁(/feedback 안내)과 세션 종료 평가 배너를 화면에서 가려 줍니다. 명령이 없는 순수 표시 필터 플러그인입니다.

동작 방식

서버가 패널 행을 클라이언트로 보내기 직전에 해당 문구 행만 공백으로 바꿉니다(server_filter_rows 훅). Claude 에 키를 주입하지 않으므로 세션 상태에 영향이 없고, 플러그인을 끄면 문구가 다시 보입니다.

켜고 끄기

별도 명령이 없습니다. :plugins 관리 팝업에서 켜고 끕니다.

끄기 · 제거

:plugins 관리 팝업에서 끄거나(가역), pytmuxlib/plugins/claude-disable-feedback/ 디렉토리를 지우면(delete-to-disable) 명령·자동완성·메뉴 어디에도 나타나지 않고 조용히 사라집니다. 코어는 영향받지 않습니다. ← 플러그인 개요

설정 파일

pytmux.conf 로 동작을 바꾸거나, 통합 설정 팝업 :settings 로 카테고리별로 조정합니다. :settings 에서 바꾼 파일 설정은 설정 파일에 자동으로 되쓰기 되고 (주석·바인딩 보존), 서버 옵션(window-size·vt-parser· automatic-rename 등)은 서버가 opts.json 에 영속합니다 — 어느 쪽이든 다시 켤 때 유지됩니다.

설정 파일 위치와 형식

설정 파일은 ~/.config/pytmux/config, ~/.pytmux.conf, 또는 환경변수 PYTMUX_CONFIG 로 지정한 경로에서 읽습니다(pytmux.conf.example 참고). 여러 곳에 있으면 PYTMUX_CONFIG → $PYTMUX_HOME/config → ~/.config/pytmux/config → ~/.pytmux.conf 순으로 먼저 있는 하나만 씁니다. 각 줄은 set <이름> <값> 형식이고, source-file 로 다시 읽습니다. 현재 값은 show-options 로 확인합니다. 실행 중에 바꿀 때는 명령 프롬프트에서 set 을 빼고 옵션 이름을 그대로 칩니다(예: :strip-box-drawing off, :lang en) — :set … 이 먹는 것은 mouse·prefix·mode-keys 처럼 일부뿐이라, 가장 확실한 방법은 설정 팝업(:settings)에서 고르는 것입니다.

통합 설정 팝업 · 언어

:settings(별칭 config·preferences)는 옵션을 표시 · 입력 · 동작 · 상태줄 · 고급 · 키 카테고리(+ 플러그인의 Claude 카테고리)로 묶어 보여 줍니다. 여기서 바꾼 값은 파일 설정이면 설정 파일에, 서버 옵션이면 서버 opts.json 에 각각 자동 반영됩니다. UI 언어는 lang ko|en (별칭 language)으로 즉시 바꾸며, 선택은 클라이언트에 저장됩니다.
